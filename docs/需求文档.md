## 一、项目背景与愿景

### 1.1 项目定位
本项目定位为**轻量级检索增强生成（RAG）知识库的最小可行产品（MVP）**，核心目标是验证RAG技术相较于传统大模型单次文档输入模式在知识沉淀、精准检索和可追溯回答三大维度的核心价值。

### 1.2 核心价值主张
| 维度 | 传统模式痛点 | RAG知识库解决方案 |
|------|-------------|------------------|
| 知识复用 | 每次提问需重新上传文件 | 文档一次入库，持续复用 |
| 信息获取 | 海量文档手动筛选效率低 | 向量检索精准定位相关段落 |
| 回答可信度 | 回答无来源追溯，易编造 | 来源标注保障真实可验证 |

### 1.3 成功标准
- 文档上传 → 解析切片 → 向量存储 → 检索生成 **全链路可正常运行**
- 回答100%基于知识库内容，无编造成分
- 支持多轮连续对话，语义上下文连贯

## 三、详细功能需求

### 3.1 文档上传管理页

#### 3.1.1 文件上传功能
- **支持格式**：`.pdf`、`.docx`、`.md`、`.txt`
- **格式校验**：上传时拦截非允许格式，提示「仅支持PDF、Word、Markdown、TXT格式」
- **文件大小限制**：单文件不超过50MB（预留扩展点）
- **并发处理**：支持多文件同时上传

#### 3.1.2 处理状态管理
| 状态 | 触发条件 | 用户可见信息 |
|------|---------|-------------|
| 处理中 | 文件上传成功，开始解析 | 显示进度提示 |
| 处理完成 | 解析切片入库完成 | 显示完成标记 |
| 处理失败 | 解析或入库异常 | 显示错误原因（预留） |

#### 3.1.3 文档列表
- **字段**：文档名称、上传时间、处理状态
- **排序**：按上传时间倒序（最新在前）
- **刷新**：支持手动刷新查看最新状态

### 3.2 智能问答页

#### 3.2.1 对话交互
- **输入**：单行/多行文本输入框
- **发送**：支持回车发送或点击按钮发送
- **展示**：消息流形式，用户消息右对齐，AI回答左对齐
- **历史**：保存当前会话的对话历史

#### 3.2.2 回答生成规则（强制约束）
```
规则1：内容约束
  - 所有回答必须100%基于知识库存储内容
  - 严禁调用大模型自身知识编造答案
  - 当Top3召回片段相似度均过低时，返回固定提示语

规则2：来源标注
  - 回答末尾必须附带参考来源信息
  - 格式：「参考来源：{文档名称} - 第{段落序号}段」
  - 多来源时逐一列出

规则3：拒答机制
  - 当知识库中无匹配内容时，固定返回：
    「当前知识库中无相关内容，无法解答」
```

#### 3.2.3 多轮对话
- 自动继承历史对话的上下文语义
- 每次回答基于「用户当前问题 + 历史对话摘要 + 召回片段」生成
- 无需用户重复说明前文背景

## 四、非功能需求

### 4.1 性能需求
| 指标 | 目标值 | 说明 |
|------|-------|------|
| 文档解析速度 | ≤5秒/份 | 10页以内PDF |
| 检索响应速度 | ≤2秒/次 | Top3召回 |
| 首屏加载时间 | ≤3秒 | 页面初始化 |

### 4.2 可用性需求
- 支持本地一键部署运行
- 无需额外部署独立服务（向量库、数据库等）
- 数据持久化存储于本地磁盘

### 4.3 扩展性需求
- 模块化设计，支持功能扩展
- 预留大模型API接口，可灵活切换（豆包、通义千问等）
- 预留文档格式扩展接口

## 五、数据流转设计

### 5.1 文档生命周期

```
上传文件 → 格式校验 → 内容解析 → 文本切片 → 向量化 → 入库存储
    │          │          │          │         │        │
    ▼          ▼          ▼          ▼         ▼        ▼
  文件池    拒绝/通过   纯文本    段落块     向量    ChromaDB
```

### 5.2 问答生命周期

```
用户提问 → 向量化 → 相似度检索 → Top3召回 → Prompt拼装 → 大模型生成 → 答案输出
   │         │         │          │          │            │            │
   ▼         ▼         ▼          ▼          ▼            ▼            ▼
 文本输入  query_emb  余弦相似度  片段列表  标准化格式   API调用     含来源答案
```
